## 快捷键

- `Ctrl + P` : 预览
- `Ctrl + O` : 详情
- `Ctrl + Shift + C` : 复制路径
- `Alt + Shift + C` : 使用vscode编辑

## 功能关键字

- `shutdown` : 关机
- `reboot` : 重启
- `uninstall` : 卸载
- `hosts` : 编辑host文件
- `Folder: game` : 搜索文件夹名中有game的文件夹
- `Video : film` : 搜索视频名中有film的视频文件
